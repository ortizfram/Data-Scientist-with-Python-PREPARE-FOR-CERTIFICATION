```py

                y="horsepower", 
                data=mpg,
                kind="scatter")

# Print type

```

## 游붌 Adding a title to a FacetGrid object
```py
# Create scatter plot
g = sns.relplot(x="weight", 
                y="horsepower", 
                data=mpg,
                kind="scatter")

# Add a title "Car Weight vs. Horsepower"
g.fig.suptitle("Car Weight vs. Horsepower")

# Show plot
plt.show()
```
