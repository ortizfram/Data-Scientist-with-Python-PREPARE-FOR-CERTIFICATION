## 游붌 Creating subplots with col and row
```py
# Change to make subplots based on study time
sns.relplot(x="absences", y="G3", 
            data=student_data,
            kind="scatter")

# Show plot
plt.show()
```

<img src="https://user-images.githubusercontent.com/51888893/208238843-82d7ab9a-c945-471b-85b2-42e3365accf1.png" width=400px>

```py

sns.relplot(x="absences", y="G3", 
            data=student_data,
```

## 游붌 Creating two-factor subplots
```py
# Create a scatter plot of G1 vs. G3
sns.relplot(x='G1', y='G3',
            data= student_data, kind='scatter')
            


# Show plot
plt.show()
```

<img src="https://user-images.githubusercontent.com/51888893/208239400-9db93c8a-a35a-494d-8d53-325f571eed89.png" width=400px>

```py
# Adjust to add subplots based on school support
sns.relplot(x="G1", y="G3", 
            data=student_data,
            kind="scatter",
            col='schoolsup',
            col_order=['yes','no'])

# Show plot
plt.show()
```

<img src="https://user-images.githubusercontent.com/51888893/208239525-36a5e547-29b1-426f-94a2-3a02879ee0fc.png" width=400px>

```py
# Adjust further to add subplots based on family support
sns.relplot(x="G1", y="G3", 
            data=student_data,
            kind="scatter", 
            col="schoolsup",
            row='famsup',
            col_order=["yes", "no"],
            row_order=["yes", "no"]
            )

# Show plot
plt.show()
```

## 游붌 Changing the size of scatter plot points
```py
# Import Matplotlib and Seaborn
import matplotlib.pyplot as plt
import seaborn as sns

#size
# Create scatter plot of horsepower vs. mpg
sns.relplot(x="horsepower", y="mpg", 
            data=mpg, kind="scatter", 
            size="cylinders",
            hue="cylinders")

# Show plot
plt.show()
```
