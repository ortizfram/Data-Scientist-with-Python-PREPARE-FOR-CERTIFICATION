## 👽 Filtering joins
#### 📚 semi join
❓ equals to inner join, returns only left and not from the right, no duplicates

👞 1
```py
# Merge 2 tablws w/ inner
genre_tracks = genres.merge(top_tracks, on='gid')
```
👞 2
```py
# .isin() returning bool
top_genres= genres[genres['gid'].isin(genres_tracks['gid'])]
```
